# 选择题

1. 特征：顺序，全篇
2. 类型
   1. 补全
   2. 问答
3. 步骤：
    1. 题干 筛选关键词
    2. 选项筛选定位词
    3. 排除：
       1. 多
       2. 错
          1. 以偏概全
          2. 偷换概念
          3. 断章取义
       3. 混
       4. 少
4. 原则：
   1. 同性相斥
   2. 异性相吸
   3. 绝对概念
   4. 陌生词原则
   5. 相近干扰原则

# 功能图题

1. 特征： 1至2段
2. 步骤： 
   1. 观察： 
      1. 图型结构，
      2. 各部件位置，
      3. 图形，
      4. 形状，
      5. 大小，
      6. 颜色
   2. 根据图中所给的英文注释，定位原文
   3. 根据结构关系，写出正确答案
